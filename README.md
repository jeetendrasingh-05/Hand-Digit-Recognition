#  Handwritten Digit Recognition using Machine Learning

##  Overview

This project implements a Handwritten Digit Recognition system using the MNIST dataset. The model is trained to classify handwritten digits (0–9) from grayscale images using machine learning techniques. It demonstrates the complete workflow from data preprocessing to model evaluation and prediction.



##  Features

 Classifies handwritten digits (0–9)
 Image preprocessing and normalization
 Machine learning model training and evaluation
 Predicts handwritten digits with high accuracy
 Visualizes sample images and model predictions



##  Tech Stack

Python
NumPy
Pandas
Matplotlib
Scikit-learn
TensorFlow / Keras
Jupyter Notebook



##  Dataset

**MNIST Handwritten Digits Dataset**

- 70,000 grayscale images
- 28 × 28 pixel images
- 10 output classes (Digits 0–9)



## ⚙️ Project Workflow

1. Load the MNIST dataset
2. Preprocess and normalize image data
3. Train the machine learning model
4. Evaluate model performance
5. Predict handwritten digits
6. Visualize predictions and results



##  Results

 Successfully classified handwritten digits with high accuracy.
Evaluated the model using test data and prediction metrics.
Visualized actual and predicted digit classifications.
